# 数据分析样本说明

## 数据文件位置

`data_sample/user_sentiment_sample.csv`

## 数据来源

本数据为模拟数据，模拟茶百道在开发新品前，从小红书、抖音、外卖评价等公开平台采集的消费者对茶饮口感的真实反馈。

## 数据字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 用户ID | 文本 | 匿名化用户标识 |
| 评价内容 | 文本 | 用户原始评价 |
| 情绪标签 | 文本 | 积极/中性/消极（基于NLP初步标注） |
| 口感关键词 | 文本 | 从评价中提取的口感词 |
| 产品提及 | 文本 | 提及的茶百道具体产品 |
| 省份 | 文本 | 用户所在省份 |

## 数据用途

本数据用于验证方案中"精准识人"模块的数据采集与处理逻辑，展示团队具备：
1. 多源数据采集能力
2. 消费者评价结构化处理能力
3. 口感关键词提取与情感标注能力
